# Analyzing an Unexplored Layer of Digital Audio Forensics with Spectrogram-Based Steganography and Detection

Python script that allows users to embed text into an audio file through spectrograms. With a command-line interface, users are able to hide hidden messages into audio files.
Users can choose to hide text or images.
Saves spectrograms as images to view.
Detects possible anomalies from audio files (If they have been embedded with a secret message).
